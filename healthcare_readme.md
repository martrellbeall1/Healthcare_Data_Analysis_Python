# Healthcare Data Analysis & Visualization (Python)

## Overview
This project demonstrates data cleaning, exploratory data analysis (EDA), and visualization using a healthcare dataset. The goal is to analyze patient health data and identify trends that support data-driven decision-making.

## Tools & Technologies
- Python (Google Colab)
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
- Source: Pima Indians Diabetes Dataset
- Format: CSV
- Description: Contains patient health metrics used to analyze diabetes outcomes

## Key Tasks Performed
- Data cleaning (handling missing values, removing duplicates)
- Exploratory Data Analysis (EDA)
- Data transformation and preprocessing
- Statistical analysis of healthcare variables
- Data visualization (heatmaps, distributions, scatterplots)

## Key Insights
- Identified correlations between patient health variables
- Observed distribution patterns in glucose levels and BMI
- Analyzed relationships between age and health indicators
- Demonstrated how data supports healthcare decision-making

## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Dataset loads directly from a public URL
